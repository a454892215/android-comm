 Flutter动画的时间核心是 AnimationController，而 AnimationController 基于 Ticker 驱动。
 AnimationController 一般配合 Tween 和 Curve 使用，一起构建动画。AnimationController 是 Animation<double> 的子类
 AnimationController：设置动画是否循环，是否反向，动画时间。控制动画的播放，停止，并且提供了动画在播放时间内，默认从0到1线性变化的时间进度映射值。
 Curve 是用来改变动画的播放速率的，默认是线性变化。
 Tween是基于AnimationController提供的时间进度映射值，进行二次转换成动画需要的实际动画值。
 AnimationController的本质是一个时间比例发生器：默认是在在指定时间内，让值从0到1匀速变化。
 系统封装好的常见的 Curve： Curves.linear，Curves.easeInOut，Curves.easeOut
 系统封装好的常见的 Tween： Tween<double> ,Tween<Offset> , ColorTween, SizeTween, DecorationTween

AnimatedBuilder 内部则封装了Animation，AnimatedBuilder是一种 Widget
Transition 是 Flutter 提供的一种 Widget。它本质是 AnimatedWidget 的子类，内部已经封装了 Animation，你不用自己写 AnimatedBuilder。

 隐式动画（Implicit Animation）：然后把动画直接为结束状态（只需要设定动画时间，和Curve），动画的中间状态可以自动完成。
 优点：简单，适合常规属性动画。
 常用组件：
 Transition
 AnimatedBuilder（使用中，需要传递animation对象）
 AnimatedContainer
 AnimatedOpacity
 AnimatedPositioned
 AnimatedAlign
 AnimatedPadding

 显式动画（Explicit Animation）：需要开发者手动创建 AnimationController 和 Animation，并管理动画状态。
 优点：灵活、可控，适合复杂动画。
 常用组件：
 AnimationController
 Tween / ColorTween / IntTween
 AnimatedBuilder
 示例：
 AnimationController _controller = AnimationController(
   vsync: this,
   duration: Duration(seconds: 2),
 );
 Animation<double> animation = Tween(begin: 0, end: 100).animate(_controller);

完全自定义flutter动画：自己计算好每一帧的UI相关参数，然后让屏幕刷新。
  真正“按屏幕帧率刷新”的只有两类：Ticker / AnimationController 和 SchedulerBinding
  Ticker = 官方推荐的、最安全、最干净的方式
