volatile关键字：
   Java中 多线程共享的成员变量 为了提高执行速度，在每个线程的内部也有自己的私有缓存拷贝（L1/L2缓存），
   所以线程读取或共享的成员变量的时候，可能直接读私有缓存拷贝 而不是堆内存的共享的成员变量，
   导致虽然其他线程更新了共享成员变量，但是后面的线程可能不能及时使用最新共享成员变量。

Java 中的基本类型和引用类型
   1. 基本类型：没有自己的成员函数和成员变量 ，局部变量一般保存在栈内存，成员变量保存在堆内存，不需要通过new 来创建对象，一般是直接赋值。
              基本类型传递参数，传递的是值的副本
   2. 引用类型：一般都是使用class关键词声明的类型，有自己的成员变量和成员函数，需要通过new 来创建对象
              引用类型传递参数，传递的是引用指向的对象
